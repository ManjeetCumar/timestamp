FreeCodeCamp API Projects: Timestamp Microservice

User stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date(example: February 27,2018)
  2. If it does, it returns both the Unix timestamp and the natural language form of the date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

http://localhost:3000/datevalues/February%2027,%202018

http://localhost:3000/datevalues/15171

Example output:

{"unix":"15171","natural":"January 01, 1970"}
